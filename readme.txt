0. ВАЖНОЕ ПРИМЕЧАНИЕ
--------------------------------------------------------------------------------
- Для работы расширения необязательно поднимать сервер с нуля — уже есть
  работающий сервер (83.242.96.235) с установленной англоязычной базой
  вопросов StackOverflow и всеми необходимыми сервисами (оплачен до конца
  июня 2026 года). Для работы плагина знание пароля от сервера не требуется
  — достаточно просто загрузить распакованное расширение в браузере
  (см. п. 5.4) и включить отслеживание. Доступ к самому серверу (по SSH)
  предоставляется по запросу (пароль изменён).
- Эмбеддинги вопросов (векторные представления для поиска похожих вопросов)
  были созданы одним из участников проекта.
- Изначальная версия плагина была создана другим участником проекта. В той
  версии отсутствовали: дедупликация, блок рекомендаций, блок похожих
  вопросов и кастомные CSS-селекторы для некоторых образовательных сайтов.
  Все перечисленные функции были добавлены в текущую версию.

1. НАЗНАЧЕНИЕ
--------------------------------------------------------------------------------
Браузерное расширение (Chrome MV3) + серверная часть для отслеживания
посещения образовательных сайтов и отображения релевантных вопросов
со Stack Overflow на основе содержимого страницы.

2. СОСТАВ ПРОЕКТА
--------------------------------------------------------------------------------
- плагин/          — Chrome-расширение (manifest v3)
- сервер/          — Серверная часть (Flask)
  - log_server.py               — сервер логов визитов (:5002)
  - services/vector_service/    — сервис векторного поиска (:5006)
  - init_fts.py                 — инициализация FTS-индекса SQLite
  - deploy.sh                   — скрипт развёртывания на Ubuntu Server
- сбор данных/     — ETL-конвейер для наполнения базы вопросов
  - ru_scraper.py              — парсинг ru.stackoverflow.com (Selenium)
  - enricher.py                 — загрузка ответов/комментариев через API
  - fetch_pyquestions_eng.py    — загрузка англоязычных вопросов

3. НЕОБХОДИМОЕ АППАРАТНОЕ ОБЕСПЕЧЕНИЕ
--------------------------------------------------------------------------------
- Сервер: минимум 2 ядра CPU, 4 ГБ ОЗУ, 20 ГБ дискового пространства
  (рекомендуется: 4 ядра, 8 ГБ ОЗУ для работы sentence-transformers)
- Клиент: любой компьютер с установленным браузером Chrome/Edge (Chromium)

4. НЕОБХОДИМОЕ СИСТЕМНОЕ И ПРИКЛАДНОЕ ПО
--------------------------------------------------------------------------------
4.1. Для сервера:
  - ОС: Ubuntu 20.04+ / Debian 11+
  - PostgreSQL 14+
  - Python 3.10+
  - Зависимости Python (устанавливаются автоматически deploy.sh):
    - Flask==3.0.0
    - gunicorn==21.2.0
    - sentence-transformers==2.2.2
    - sqlite-vec==0.1.5
    - psycopg2

4.2. Для сбора данных (ETL):
  - Python 3.10+
  - Selenium + ChromeDriver (для ru_scraper.py)
  - Библиотеки: requests, beautifulsoup4, sentence-transformers, sqlite3

4.3. Для плагина (клиент):
  - Браузер Chrome 88+ / Edge 88+ / любой Chromium-браузер
  - Node.js + npm (только для запуска тестов)

5. ЗАПУСК ПРОГРАММЫ
--------------------------------------------------------------------------------

5.1. Запуск сервера логов (log_server):
  cd plugin+serverV26-copy/сервер
  pip install flask psycopg2
  python log_server.py
  → Сервер будет доступен на порту 5002

5.2. Запуск сервиса векторного поиска (vector_service):
  cd plugin+serverV26-copy/сервер/services/vector_service
  pip install -r requirements.txt
  python app.py
  → Сервис будет доступен на порту 5006

  При первом запуске потребуется база данных q.sqlite
  (создаётся скриптами из папки "сбор данных").

5.3. Автоматическое развёртывание на сервере:
  cd plugin+serverV26-copy/сервер
  # Поместить pg_backup.sql и q.sqlite в текущую директорию
  sudo bash deploy.sh

5.4. Установка Chrome-расширения:
  1. Открыть Chrome → chrome://extensions
  2. Включить "Режим разработчика"
  3. Нажать "Загрузить распакованное расширение"
  4. Выбрать папку plugin+serverV26-copy/плагин
  5. Готово. Расширение появится в панели расширений.

5.5. Сбор данных (ETL) — наполнение базы вопросов:
  # Парсинг ru.stackoverflow.com (тег python):
  cd сбор\ данных
  pip install selenium beautifulsoup4
  python ru_scraper.py

  # Загрузка ответов и комментариев через Stack Exchange API:
  python enricher.py

  # Загрузка англоязычных вопросов:
  python fetch_pyquestions_eng.py

5.6. Инициализация FTS-индекса (после создания БД):
  cd plugin+serverV26-copy/сервер
  python init_fts.py /путь/к/q.sqlite

6. КОНФИГУРАЦИЯ
--------------------------------------------------------------------------------
- API_BASE (background.js): адрес сервера логов (по умолчанию http://83.242.96.235:5002)
- SIMILARITY_URL (background.js): адрес сервиса векторного поиска
- DATABASE_URL (log_server.py): строка подключения к PostgreSQL
- DB_PATH (app.py): путь к SQLite-базе с вопросами

7. ПРОВЕРКА РАБОТЫ
--------------------------------------------------------------------------------
После запуска всех компонентов:
- Открыть любой образовательный сайт (stackoverflow.com, github.com, stepik.org и др.)
- Расширение автоматически извлечёт текст страницы и покажет оверлей
  с похожими вопросами из базы StackOverflow
- Нажать на иконку расширения → откроется popup с кнопками управления
  и рекомендациями на основе истории просмотров
